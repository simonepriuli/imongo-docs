# validate a schema

To validate a schema you can chose useSafeValidation() or useUnsafeValidation()

useSafeValidation() throws an error if the schemas doesn't match, if it does match it return the same object passed as parameter.

```js filename="Safe schema validation" {3} copy
const object = { name: "Simone" };

const validatedObject = db.useSafeValidation(object);
```

useUnsafeValidation() return either the same object (the schemas match), or an error object (the schemas doesnt'match).

```js filename="Unsafe schema validation" {3} copy
const object = { name: "Simone" };

const validatedObject = db.useUnsafeValidation(object);
```
