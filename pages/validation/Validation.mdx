# Validate a schema with Zod

**You can read the Zod documentation [here](https://zod.dev/).**

To validate a schema you can chose useSafeValidation() or useUnsafeValidation()

useSafeValidation() throws an error if the schemas doesn't match, if it does match it return the same object passed as parameter.

### Safe validation
```js filename="Safe schema validation" {3} copy
const object = { name: "Simone" };

const validatedObject = db.useSafeValidation(object, 'collection name');
```

### Unsafe validation
useUnsafeValidation() return either the same object (the schemas match), or an error object (the schemas doesnt'match).

```js filename="Unsafe schema validation" {3} copy
const object = { name: "Simone" };

const validatedObject = db.useUnsafeValidation(object, 'collection name');
```

### Chaining
Note that in both useSafeValidation and useUnsafeValidation the collection name is optional if you chain the method the collection method like this.
```js filename="Unsafe schema validation" {3} copy
const object = { name: "Simone" };

await db.collection('users').insertOne(db.useSafeValidation(object));
```
